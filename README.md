# Simple Interest Calculator

## Description
This repository contains a simple script to calculate simple interest based on user input. It is part of the final project for the Git and GitHub course.

## Formula
The formula used to calculate Simple Interest is:
**Simple Interest = (Principal * Rate * Time) / 100**

* **Principal (p)**: The initial amount of money borrowed or invested.
* **Rate (r)**: The annual interest rate (in percentage).
* **Time (t)**: The time period in years.

## Usage
To use this calculator, you need to provide the Principal, Rate of interest, and Time period when prompted by the script.
